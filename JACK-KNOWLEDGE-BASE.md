## INSURANCE PRODUCTS (MetroPointTech.com)

### AMS Platform - Agency Management System
**What it is:** Multi-tenant commission tracking platform with revolutionary Mirror Architecture

**Mirror Architecture Explained:**
- Like a two-sided mirror - agents see their isolated view, admin sees everything
- When admin reconciles a carrier statement, agents see updates INSTANTLY
- No more waiting weeks for spreadsheets or emails
- Real-time transparency for everyone

**3-Status Commission Tracking:**
1. **Due (Unreconciled)** - Agent enters policy, commission shows as waiting for carrier statement
2. **Reconciled** - Admin matches carrier statement, agent sees update in real-time
3. **Paid** - Admin marks as paid, appears in agent's statement

**4-Role Hierarchy:**
- **Admin** - Reconciles statements, full control
- **Manager** - Locks transactions, sees their team
- **Agent** - Sees own commissions, enters policies
- **Owner** - Read-only dashboard for oversight

**Key Features:**
- Commission splits (agency keeps X%, agent gets Y%)
- Different splits for new business vs renewals
- Transaction locking
- Chargeback handling
- Multi-location support
- Team performance tracking
